# Drop ME

Drop ME is a rural-first ride service MVP for affordable and safer local travel.

This starter app demonstrates:

- passenger ride booking
- verified local driver selection
- low driver commission positioning
- guardian ride sharing
- SOS alert flow for guardian and nearest police station
- rural-friendly messaging around low internet and phone-first use

## Run locally

Open `index.html` in any modern browser.

No build tools or package installation are required.
